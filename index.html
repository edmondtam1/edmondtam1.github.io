I co-created Jade, an open source framework for deploying JAMstack apps, built with Node.js & Amazon Web Services.

Jade is an open-source framework that automates JAMstack app deployment via GitHub and AWS services.

Jade provisions infrastructure needed for developers to automatically build multiple apps on EC2, host on S3, distribute on CloudFront and instantly invalidate the CDN with Lambda. Jade is written using Node.js, Docker, React and Express.
